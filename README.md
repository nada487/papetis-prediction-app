# PAPETIS — Application de Prévision et Pilotage Logistique

Application décisionnelle développée en Python (Streamlit) pour l'analyse prévisionnelle et la simulation de scénarios de ventes (Option Innovation & AMOA).

## 🚀 Guide d'Installation Rapide

1. **Cloner le projet :**
```bash
git clone [https://github.com/VOTRE_COMPTE/papetis-prediction-app.git](https://github.com/VOTRE_COMPTE/papetis-prediction-app.git)
cd papetis-prediction-app

Pour installer les dépendances:
Bash
pip install streamlit pandas numpy plotly

Lancer l´application:
streamlit run app.py
